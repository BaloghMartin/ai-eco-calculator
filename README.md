# AI Eco Calculator (MI Környezeti Hatás Kalkulátor)

🌍 *[🇭🇺 Kattints ide a magyar verzióért! (Click here for the Hungarian version)](#-magyar-verzió)*

This repository contains the source code for an interactive web widget (HTML/JS) that estimates the ecological footprint (energy and water consumption) of Large Language Model (LLM) prompts and queries (e.g., ChatGPT, Llama, Claude). 

The project was originally created for data journalism and demonstration purposes to illustrate the real-world environmental impact of artificial intelligence.

## ⚙️ How It Works & Technology

The calculator runs entirely client-side using the following methodology:
* **Processing:** Based on input parameters, it uses empirical consumption constants to calculate estimated energy (kWh) and water (ml) footprint.
* **Hash Technology:** To uniquely and anonymously identify queries, the system uses the highly fast and secure **FNV-1a** hashing algorithm.
* **Tech Stack:** HTML5, CSS3, Vanilla JavaScript (no external dependencies).
